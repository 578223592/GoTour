## 项目目的

- 用于帮助大家急速入门Go语言
- 自己日后复习

## 特点

- 精简：能用代码清晰简单的表示的就不赘述
- 浅显：从最佳实践出发，避免冗余的知识

## 前置要求

需要你拥有至少一门其他语言的基础，并拥有一些关于并发、多线程的知识，这样可以更快速的上手。

如果你没有相关的知识基础，那么你可以：

- 提一个issue来询问我，我会给你一些建议或者直接给出答案
- 使用搜索引擎搜索关键词，大概了解相关概念即可

> 本人之前用C++比较多，也使用过Java。因此可能部分概念会用C++或者Java来类比，如果有这两门语言的知识最好，没有也不影响。
