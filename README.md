## 项目目的

- 用于帮助大家急速入门Go语言

## 前置要求

需要你拥有至少一门其他语言的基础，并拥有一些关于并发、多线程的知识，这样可以更快速的上手。

如果你没有相关的知识基础，那么你可以：

- 提一个issue来询问我，我会给你一些建议或者直接给出答案
- 使用搜索引擎搜索关键词，大概了解相关概念即可

> 本人之前用C++比较多，可能部分概念会用C++来类比
